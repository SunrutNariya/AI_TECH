#  Smart Calculator

This is a simple Python-based calculator that:
- Performs +, -, *, /, %
- Logs results to a file
- Tells whether the numbers are Even/Odd and Positive/Negative

##  Flowchart

 ↓

 ↓

 ↓

 ↓

 ↓
